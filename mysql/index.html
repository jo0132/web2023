MYSQL

MySQL은 오픈 소스 관계형 데이터베이스 관리 시스템(RDBMS)입니다.

MySQL은 다중 사용자, 다중 스레드 RDBMS로서, 데이터의 안정성과 보안성을 제공합니다. MySQL은 대부분의 운영 체제에서 작동하며, 웹 애플리케이션 개발, 데이터 분석, 빅 데이터, 클라우드 기반 애플리케이션 등에 많이 사용됩니다.

MySQL은 SQL(Structured Query Language)을 사용하여 데이터를 관리합니다. SQL은 데이터베이스에 저장된 데이터를 조작하기 위한 표준적인 언어로서, 데이터를 검색, 삽입, 업데이트, 삭제하는데 사용됩니다. MySQL은 이러한 SQL 문을 이용하여 데이터를 쿼리하고, 데이터의 무결성을 유지하며, 데이터베이스를 관리합니다.

MySQL은 사용이 간편하며, 대용량 데이터베이스를 처리할 수 있는 고성능 기능을 제공합니다. 또한, 오픈 소스로서, 다양한 개발자들이 지속적으로 개발 및 유지보수하고 있어, 사용자들이 다양한 문제들을 해결할 수 있는 방법을 제공합니다.

MySQL은 PHP 스크립트 언어와 상호 연동이 잘 됨녀서 오픈소스로 개발된 무료 프로그램입니다. 그래서 홈페이지나 쇼핑몰(워드프레스, CAFE24, 그누보드, 제로보드)에 일반적으로 웹개발에 널리 사용하고 있습니다.

🦄테이블 데이터

INSERT INTO 테이블이름(필드명) VALUE
INSERT INTO member(youEmail, youName, youpass, youBirth, youAge, regTime) VALUES("jo0132@naver.com","노지영","1234","19950323","29","03140316");

테이블 불러오기

SELECT 필드명 FROM 테이블명 WHERE 조건

전체 데이터 불러오기

SELECT * FROM member2

MemberID가 1번인 경우 불러오기

SELECT * FROM member2 WHERE MemberID = 1;

MemberID가 1번이 아닌 경우 경우 불러오기

SELECT * FROM member WHERE MemberID <> 1;

MemberID가 5~10번 불러오기

SELECT * FROM member WHERE MemberID >=5 AND memberID <=10;

youAge가 30~40경우만 불러오기

SELECT *FROM member WHERE youAge BETWEEN 30 AND 40;

youAge가 30~40가 아닌 경우 불러오기

SELECT *FROM member WHERE youAge NOT BETWEEN 30 AND 40;

memberID가 1,5,10,15번만 불러오기

SELECT *FROM member WHERE MemberID IN(1,5,10,15);

youName 중에 "연"텍스트가 포함된 이름 불러오기

SELECT *FROM member WHERE youName LIKE '%연%';

youName 중에 "김"으로 시작하는 이름 불러오기

SELECT *FROM member WHERE youName LIKE '김%';

youName 중에 "현"으로 끝나는 이름 불러오기

SELECT *FROM member WHERE youName LIKE '%현';

youName 중에 "상"으로 들어가는 이름 불러오기(_은 자리수)

SELECT *FROM member WHERE youName LIKE '_상_';

youName 중에 "김" 또는 "이"로 시작하는 이름 불러오기

SELECT youName FROM member WHERE youName LIKE '김%' OR youName LIKE '이%';

이름만 불러오기

SELECT youName FROM member;

이름만 불러오기(내림차순)

SELECT youName FROM member ORDER BY youName DESC;

이름만 불러오기(오름차순)

SELECT youName FROM member ORDER BY youName ASC;

이름 갯수 불러오기

SELECT count(youName) FROM member;
SELECT count(*) FROM member;

이름과나이만 불러오기

SELECT youName,youAge FROM member;

30살 이상인 사람의 이름과나이만 불러오기

SELECT youName,youAge FROM member WHERE youAge > 30;

나이가 25이상이면서 김씨성을 갖고 있는 불러오기

SELECT youName FROM member WHERE youAge >= 25 AND youName LIKE '김%';

나이가 25이상이고 또는 김씨성을 갖고 있는 불러오기

SELECT youName FROM member WHERE youAge >= 25 OR youName LIKE '김%';

나이가 25이상이고 아이디가 2또는 아이디가 9인 멤버 불러오기

SELECT * FROM member WHERE (memberID = 2 OR memberID = 9) AND youAge >= 25;
SELECT * FROM member WHERE youAge >=25 AND memberID IN(2,9);

회원 데이터를 5개만 불러오기

SELECT youName FROM member LIMIT 5;
SELECT youName FROM member LIMIT 5, 5; // 첫번째 숫자가 기준으로 다음 수가 출력되는 첫 수, 두번째 숫자는 출력하고자하는 갯수

회원 이메일을 기준을 10개만 불러오기(11~20)

SELECT youEmail FROM member LIMIT 10, 10;

회원 나이중에 중복되는 것 제거된 값 출력

SELECT DISTINCT youAge FROM member;

테이블 수정하기

UPDATE 테이블명 SET 칼럼명 조건

모든 회원의 비밀번호를 1234로 수정하기

UPDATE member SET youPass = "1234";

아이디가 3번인 회원만 비밀번호를 9999로 수정하기

UPDATE member SET youPass = "1234";

아이디가 5-10번인 회원의 비밀번호를 7777로 수정하기

UPDATE member SET youPass = "7777" WHERE memberID BETWEEN 5 AND 10;

비밀번호를 1234567로 변경하고 패스워드를 내림차순으로 10개만 변경하기

UPDATE member SET youPass = "7777" WHERE memberID BETWEEN 5 AND 10;

아이디가 1-10번까지 나이를 50으로 변경하기

UPDATE member SET youAge = "50" WHERE memberID LIMIT 10;

이름기준 내림차순으로 5명 비밀번호를 9999로 변경하기

SELECT * FROM member ORDER BY youName desc;

테이블 삭제하기

DELETE FROM 테이블명 WHERE 조건

아이디가 5번인 맴버 삭제

DELETE FROM member WHERE memberID = 5;

비밀번호가 1234인 사람만 삭제

DELETE FROM member WHERE youPass = "1234";

회원 이름을 기준으로 5번재 멤버까지를 삭제

DELETE FROM member ORDER BY memberID LIMIT 5;

회원 이름을 기준으로 멤버를 삭제(2,3,6번)

DELETE FROM member ORDER BY memberID IN(2, 3, 6);
