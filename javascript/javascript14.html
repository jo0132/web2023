01. 퀴즈 이펙트 : 정답확인하기유형

정답 확인버튼을 누르면 정답이 나오는 효과입니다.

  • 데이터 저장하기 : 변수
  • queryselector() : 문서에서 특정 CSS 선택자로 요소를 찾는 DOM 메서드입니다. 이 메서드는 주어진 선택자와 일치하는 문서 내 첫 번째 요소를 반환합니다.
  • innerText : DOM 요소의(문서에서) 텍스트 내용을 가져오거나 설명하는데 사용되는 속성입니다.
  • style.속성 = "속성값" : 자바스크립트에 css스타일을 표현하는 방법입니다.
  • classList : 요소의 클래스를 추가하거나 제거하여 스타일을 설정하는 데 사용됩니다.
  • addEventListener : 자바스크립트에서 이벤트를 처리하는 함수입니다.

02. 퀴즈 이펙트 : 주관식 확인하기 유형

정답을 적으면 정답을 확인해 주는 효과입니다.

  • 데이터 저장하기 : 변수
  • 데이터 제어하기 : if문
  • trim : trim() 메소드는 문자열에서 양쪽 끝에 있는 공백을 제거하는 기능을 합니다.
  • textContent : 요소(element)의 텍스트 콘텐츠(text content)를 가져오거나 설정하는 속성(property)입니다.
  • value : 여기서는 input 요소에서 입력한 값(input value)을 가져오거나 설정하는 데 사용됩니다.

03. 퀴즈 이펙트 : 주관식(여러문제) 확인하기 유형

정답을 적으면 정답을 확인해 주는 효과입니다.

  • 데이터 저장하기 : 배열, 객체
  • 데이터 불러오기 : for문, forEach()
  • 데이터 제어하기 : if문
  • querySelectorAll : querySelectorAll 메서드는 문서 내에서 CSS 선택자에 해당하는 모든 요소를 찾아서 NodeList 객체로 반환하는 메서드입니다.
  • innerHTML : innerHTML은 요소의 내부 HTML 콘텐츠를 나타내는 속성입니다. 이 속성을 사용하면, 요소 내부의 HTML 코드를 문자열로 읽거나 변경할 수 있습니다.
